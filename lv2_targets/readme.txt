本フォルダにはレベル2用のターゲット認識器を可視化した画像群が保存されています．
レベル2用のターゲット認識器はいずれも，
Cifar10訓練データ50,000枚を用いて学習したマルチラベル認識器であり，
各ラベルごとに一枚の可視化画像が存在します．

各画像の解像度は 512x512 で，横軸が特徴量の1次元目(x1)，縦軸が特徴量の2次元目(x2)を表します．
各次元の値は -1 から 1 までであり，左端が x1 = -1 ，右端が x1 = 1 に対応します．
同様に，上端が x2 = 1 に，下端が x2 = -1 に対応します．

色は，そのピクセルに対応する特徴量が入力された時の，
対応するラベルの尤度（確率）を表しています．
黒が尤度0，白が尤度1を表します．

一つのターゲット認識器につき一つのフォルダが対応しており，
フォルダ内には各ラベルに関するターゲット認識器の可視化結果が
{ラベル名}.png という名前で保存されています．
